Using the Shopping Cart

The Shopping Cart is a highly customizable, single-step, mobile-ready checkout solution for your website. The Shopping Cart includes features such as support for discount codes, customizable fields, subscription payments, PayPal, multiple currencies and languages, customer account creation and more.

Creating Links to Shopping Cart

You can create links manually using the documentation provided below, or even easier, use the Link Builder tool.

Customization Options

Shopping Cart allows a number of customizations, such as:

  • Full control over the design, colors, logo, etc.
  • Select different layouts
  • Collect payment in multiple currencies
  • Multi-language order form
  • Payments by credit card, PayPal, Amazon Pay with others to be added in the future
  • Allow customers to save their payment method with a username and password for speedy future payments
  • Collect custom data from customers with custom fields that can be optional or required
  • Support for discount codes, either provided through the URL or directly input by the customer
  • Provide additional custom data with the payment for your reference

To set your preferences, login to your account, go to Apps> Shopping Cart. Click on the arrow next to "Launch" and select "Settings" and "Style"

Creating Links Manually

Below are several examples, you can mix and match parameters to provide the options you want.

Add a product to the cart
{{data.src}}/#/cart?product_id:1001

Add a product to the cart, specify the quantity
{{data.src}}/#/cart?product_id:1001=2

Add multiple products to the cart
{{data.src}}/#/cart?product_id:1001&product_id:1002
You can provide an unlimited number of products in a single URL

Add multiple products to the cart, specify the quantity for each
{{data.src}}/#/cart?product_id:1001=1&product_id:1002=2

Add a product to the cart, remove any existing items already in the cart
{{data.src}}/#/cart?product_id:1001&empty_cart=true

Specify a promotion code
{{data.src}}/#/cart?product_id:1001&promotion_code=xxx
You can create promotion codes in Store> Promotions

Provide customer details
{{data.src}}/#/cart?product_id:1001&name=John%20Doe&email=john@example.com

Provide referrer data
{{data.src}}/#/cart?product_id:1001&referrer=my-custom-value
Allows you to specify custom data that will help you know where the sale came from. The value you provide will be recorded with the cart and completed order.

Define the payment currency
{{data.src}}/#/cart?currency=EUR&product_id:1001
Generally you don't need to specify the currency. If not supplied, the best currency for the customer will be automatically selected and the customer is able to change the currency in the cart.

Provide custom data
{{data.src}}/#/cart?product_id:1001&this=that&more=stuff
Any parameters you specify will be stored with the cart and completed order, for your reference. You can supply up to 2048 characters of custom data.
